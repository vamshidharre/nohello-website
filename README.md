# nohello-website

A clean, single-page website about chat etiquette — why you shouldn't just say "hello" in chat without including your actual question.

## What is this?

In workplace chat tools (Slack, Teams, Discord, etc.), many people send a message like "Hi" or "Hey, are you there?" and then wait for a response before asking their actual question. This creates an awkward pause where the other person has to sit and wait.

This website explains the problem and offers a simple fix: **include your question with your greeting.**

## How to use

Just open `index.html` in a browser. No build tools, no dependencies, no framework — it's plain HTML, CSS, and JavaScript.
